# 광동댐 가뭄 예측 모델

## 프로젝트 개요

본 프로젝트는 **강원도 삼척시 광동댐 수문·기상 데이터를 활용하여 가뭄 상태를 예측**하는 데이터 분석 및 머신러닝/딥러닝 프로젝트입니다.

가뭄은 단기간에 해소되기 어려운 재난으로, 수자원 관리와 농업·생활용수 공급에 직접적인 영향을 미칩니다. 본 프로젝트는 다양한 가뭄 지수(MSWSI, SPI6)를 활용하여 **가뭄 발생 패턴을 분석하고, 모델별 예측 특성을 비교·검증**하는 데 목적을 두었습니다.

본 프로젝트는 서비스 구현이 아닌, **시계열 데이터 분석, 모델링 전략 수립, 실험 비교를 통한 분석 역량 강화**를 목표로 한 포트폴리오용 프로젝트입니다.

---

## 문제 제기

광동댐은 강원 지역의 생활·공업·농업용수를 공급하는 핵심 수자원 시설로, 최근 몇 년간 반복되는 가뭄으로 인해 안정적인 수자원 관리의 중요성이 커지고 있습니다.

가뭄 데이터는 다음과 같은 특성을 가집니다.

* 뚜렷한 **계절성 및 추세**
* 기상·수문 변수 간 **복합적인 상호작용**
* 가뭄 단계별 **심각한 클래스 불균형**

이로 인해 단일 모델만으로는 가뭄 상태를 안정적으로 예측하기 어렵습니다. 본 프로젝트에서는 **전통 시계열 모델부터 머신러닝, 딥러닝 모델까지 폭넓게 적용**하여, 각 접근법의 장단점과 한계를 비교 분석하고자 했습니다.

---

## 프로젝트 설명

본 프로젝트는 광동댐 관련 시계열 데이터를 기반으로 가뭄 지수를 예측·분류하는 구조로 진행되었습니다.

### 데이터 및 전처리

* 가뭄 지수: **MSWSI, SPI6**
* 기상·수문 관련 독립 변수 활용
* 이상치 및 분포 특성을 고려하여 **RobustScaler 적용**
* VIF 분석을 통한 다중공선성 확인 및 해석 안정성 확보

### EDA 및 분석

* Boxplot, Histogram, Countplot을 통한 분포 및 불균형 확인
* 산점도 분석을 통한 변수 간 선형·비선형 관계 확인
* Seasonal Decompose, ADF, ACF/PACF 분석을 통한 시계열 특성 파악

### 모델링

* **전통 시계열 모델**: SARIMA, SARIMAX, VAR, VARMAX
* **머신러닝 모델**: Random Forest, KNN, XGBoost, LightGBM
* **딥러닝 모델**: CNN, RNN, LSTM

각 모델에 대해 하이퍼파라미터를 조정하며 예측 성능과 학습 안정성을 비교하였고, 다중분류/이진분류 설정에 따른 성능 변화도 함께 분석했습니다.

### 결과 및 인사이트

* MSWSI는 변동성이 큼에도 불구하고 주요 가뭄 패턴을 비교적 잘 포착
* SPI6는 **심각한 클래스 불균형**으로 인해 일부 가뭄 단계 예측에 한계 존재
* 데이터 분포와 클래스 불균형이 모델 성능에 **결정적인 영향**을 미침을 확인

## 내 역할

## 담당 역할 (**Neural Network** 모델)

* **통계적 베이스라인 구축**: SARIMA/SARIMAX 모델을 통해 계절성과 외생 변수 효과를 분석하고, 장기 예측 시 평탄화(Flattening) 현상을 확인하여 비선형 모델 도입의 필요성을 도출함.
* **비선형 모델링**: XGBoost를 적용해 수문·기상 변수와 가뭄 지수 간 비선형 관계를 학습하고, 클래스 불균형 환경에서의 성능 한계를 분석함.
* **시계열 Neural Network 실험**: RNN 기반 모델에서 다중 분류와 이진 분류를 비교하며, 모델 구조보다 데이터 분포와 타깃 설계가 성능에 더 큰 영향을 미친다는 인사이트를 도출함.

---

> 본 프로젝트는 실제 운영 시스템이 아닌, 가뭄 예측 문제를 통해 시계열 분석 및 다양한 모델링 기법의 적용 가능성과 한계를 분석한 포트폴리오용 프로젝트입니다.
